## [2.4.0] - 2026-01-30

### Added - Excel Analysis Template

- **Professional Excel Dashboard**
  - 5 priority metric tiles (TRB Eligible, CAT I, Aged Items, Total, New)
  - Status distribution pie chart
  - Severity breakdown bar chart
  - Automated Power Query data loading
  - Conditional formatting (red/orange/yellow)
  - Print-ready layout

- **6 Specialized Data Sheets**
  - Complete Data, Trends, TRB Focus
  - Priority Items, Aged Items, Summary
  - Freeze panes, filters, formatting

- **Template Features**
  - Excel 2013+ compatible
  - Comprehensive Instructions sheet
  - 20 KB file size

### Added - Documentation

- EXCEL_TEMPLATE_GUIDE.md (522 lines, comprehensive)
- EXCEL_TEMPLATE_GUIDE.docx (40 KB, print-ready)
- Tools & Resources section (About tab)
- Excel template usage guide (About tab)

### Added - Tools Documentation

- **Evaluate-STIG** (preferred, CAC required)
- **SCC Tool v5.3** (public alternative)
- **Tool Selection Guidance** (decision matrix)
- **STIG Viewer v2.4.0**

### Changed - Technical Accuracy

- Evaluate-STIG repositioned as preferred tool

- Manual POA&M tracking documented
- Realistic workflow (automation + manual)

### Updated
